## Löpande uppgifter

### Dagligen/Veckovis

- [ ] Spara kvitton och verifikationer
- [ ] Notera mottagna produkter (barter)
- [ ] Dokumentera arbetsresor

### Månadsvis

- [ ] Stäm av bankkonton mot bokföring
- [ ] Bokför månadens transaktioner
- [ ] Kontrollera plattformsutbetalningar
- [ ] Beräkna eventuella valutakursdifferenser
- [ ] Momsdeklaration (om månadsredovisare)

### Kvartalsvis

- [ ] Momsdeklaration (om kvartalsredovisare)
- [ ] Periodisk sammanställning (om EU-försäljning)
- [ ] OSS-deklaration (om EU-privatpersonsförsäljning)
- [ ] Resultatuppföljning
- [ ] SGI-kontroll

### Årligen

**Januari-februari:**
- [ ] Upprätta årsbokslut för föregående år
- [ ] Sammanställ alla intäkter och kostnader
- [ ] Beräkna avskrivningar
- [ ] Hantera periodiseringsfonder (EF)

**Mars-april:**
- [ ] Förbered inkomstdeklaration
- [ ] Kontrollera kontrolluppgifter
- [ ] Granska förifylld deklaration

**Maj:**
- [ ] Lämna inkomstdeklaration (2 maj)
- [ ] NE-bilaga (EF)

**Juni-juli:**
- [ ] Årsredovisning till Bolagsverket (AB)
- [ ] Inkomstdeklaration AB (1 juli)

**Augusti-september:**
- [ ] Analysera slutskattebesked
- [ ] Justera preliminärskatt för nästa år
- [ ] Planera för eventuell kvarskatt

**Oktober-december:**
- [ ] Skatteplanering inför årsskiftet
- [ ] 3:12-beräkning (AB)
- [ ] Pensionssparande (avdragsgillt)
- [ ] Friskvårdsbidrag (AB)

## Påminnelser per bolagsform

### Enskild firma (EF)

| Tidpunkt | Uppgift |
|----------|---------|
| Månadsvis | Bokföring, avstämning |
| Kvartalsvis | Moms, periodisk sammanställning |
| 2 maj | Inkomstdeklaration + NE-bilaga |
| Löpande | Dokumentera barter, resor, avdrag |

### Aktiebolag (AB)

| Tidpunkt | Uppgift |
|----------|---------|
| Månadsvis | Bokföring, lönehantering, arbetsgivardeklaration |
| Kvartalsvis | Moms, periodisk sammanställning |
| 31 mars | Bokslut (kalenderår) |
| 30 juni | Årsredovisning till Bolagsverket |
| 1 juli | Inkomstdeklaration AB |
| December | 3:12-beräkning, utdelningsbeslut |

### Dubbel struktur (EF + AB)

| Tidpunkt | Uppgift |
|----------|---------|
| Månadsvis | Separat bokföring båda, internfakturor |
| Kvartalsvis | Separata momsdeklarationer, periodiska sammanställningar |
| 2 maj | Inkomstdeklaration privatperson + NE |
| 30 juni | Årsredovisning AB |
| 1 juli | Inkomstdeklaration AB |
| Löpande | Avstämning interntransaktioner |

## Kritiska datum att INTE missa

### Förseningsavgifter

| Uppgift | Avgift vid miss |
|---------|-----------------|
| Periodisk sammanställning | 1,250 kr |
| Momsdeklaration | 625 kr (ökar vid upprepad försening) |
| Inkomstdeklaration | 1,250 kr (kan öka) |
| Årsredovisning | Förseningsavgift + risk för likvidation |

### Viktiga gränsdatum

| Datum | Händelse |
|-------|----------|
| 2 maj | Sista dag inkomstdeklaration privatperson |
| 30 juni | Sista dag årsredovisning AB (6 mån efter bokslut) |
| 1 juli | Sista dag inkomstdeklaration AB (kalenderår) |
| 31 december | Räkenskapsårets slut, sista dag för avdragsgilla pensionsinbetalningar |
