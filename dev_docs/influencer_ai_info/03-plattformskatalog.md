## Valutakurser

### Huvudregel

Använd valutakursen den dag inbetalningen sker till bankkontot.

### Kurskällor

- Riksbankens dagskurser (officiell)
- Bankens faktiska växlingskurs (om tillgänglig)

### Praktisk hantering

```
1. Notera belopp i originalvaluta
2. Notera inbetalningsdatum
3. Hämta kurs för det datumet
4. Beräkna SEK-belopp
5. Bokför i SEK
6. Vid fakturadatum ≠ betalningsdatum: Bokför kursdifferens
```
